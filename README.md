- Employee Directory App
A responsive Employee Directory App build using React,tailwind CSS,Mock API

- Features

1) Add new employees
2) Edit existing employee details
3) Remove employees 
4) Search employees by name, department, or role
5) Responsive Design 
6) API Integration (MockAPI) for Fetch,add,remove and update Employees

- Tech Stak Used

- Frontend
    - Reactjs
    - Tailwind CSS
    - React-hot-toast
    - Axios

- Backend
    - MockAPI.io
